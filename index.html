About the Superhero Multiverse

This website is a learning project built using an HTML5 template called "Multiverse" by HTML5 UP. It has been customized to showcase a gallery of superheroes and is hosted on AWS as part of a serverless web application. The goal is to demonstrate cloud deployment, version control, and clean commits using Git and GitHub.

This project highlights key AWS services such as S3 for static website hosting, Lambda for serverless functions, and API Gateway for dynamic functionality, while integrating a beautifully designed frontend.
